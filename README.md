# Unity Reference Game with Huawei Mobile Services Plugin for Junction

## Overview
kits used in this demo:
* In-App-Purchases
* Ads kit
* Account Kit
* Game Service

# Project Setup

## HMS Unity Plugin Integration

Create IAP products with product ID NoAds, Premium, 100Coin.
